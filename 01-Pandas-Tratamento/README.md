## Pandas – Tratamento de Dados (Churn)

## Objetivo
Realizar o tratamento completo de uma base de dados no formato JSON, aplicando técnicas de limpeza, normalização e preparação dos dados para análises exploratórias e modelagem preditiva.


### Foco do estudo
- Manipulação de DataFrames
- Normalização de dados aninhados (JSON)
- Tratamento de valores nulos e vazios
- Correção de tipos de dados
- Preparação da base para análise e modelagem

### Dataset
- Base de dados de Churn de Clientes  
  [dataset-telecon.json](https://raw.githubusercontent.com/Sofia-bispo/Jornada-Data-Science/main/01-Pandas-Tratamento/dataset-telecon.json)

## Resultado
Base de dados limpa, estruturada e pronta para análises e modelos de Machine Learning.

### Contexto
Estudo desenvolvido a partir de um curso da **Alura**, com organização e adaptações próprias para fins de aprendizado e portfólio.
